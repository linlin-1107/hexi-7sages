# Hexi Sage Council — 和熹智囊团

多智能体协作决策框架，单模型激活7种推理风格人设，零成本实现多视角分析。

## 快速安装

运行安装脚本，自动将人设文件复制到 OpenClaw 工作空间：

```bash
bash scripts/install.sh
```

脚本会：
1. 自动检测工作空间路径（`~/.qclaw/workspace`）
2. 备份已有人设文件（如有）
3. 复制 5 个人设配置文件
4. 提示下一步操作

安装后运行 `openclaw gateway restart` 生效。

## 文件结构

```
persona/
├── IDENTITY.md  — 智囊团身份定义（8位角色+深度设定）
├── SOUL.md      — 价值观、行为准则、边界
├── AGENTS.md    — 工作空间规则、记忆管理、安全红线
├── USER.md      — 用户画像模板（安装后需填写）
└── MEMORY.md    — 长期记忆模板（安装后需填写）
```

## 安装后配置

1. 编辑 `USER.md` — 填写用户名称、时区、工作领域、行为准则
2. 编辑 `MEMORY.md` — 添加长期记忆内容
3. 可选：编辑 `IDENTITY.md` 自定义智囊团成员或角色设定

**工作流程**：用户提问 → 和熹理解需求 → 召集相关智囊 → 多视角分析 → 汇总裁决 → 输出方案

## 核心理念

- **单模型多视角**：不依赖多个 LLM 实例，通过 Prompt Engineering 在单一模型内激活不同推理风格
- **历史人物人设**：用古代贤士替代现代名人 Agent，规避法律风险，增强文化辨识度
- **零成本运行**：无需额外 API 调用，所有智囊共享同一模型的上下文窗口
- **渐进式裁剪**：非关键问题只召集2-3位相关智囊，重大决策才全员集结
