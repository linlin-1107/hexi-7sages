# 👑 Hexi 7 Sages — 和熹 AI 智囊团

> 以东汉女政治家**邓绥**为原型的多智能体协作决策框架，专为 [OpenClaw](https://openclaw.ai) 设计。

## 简介

和熹（Hexi）是一个融合古今智慧的 AI 智囊。每当用户提问，她会召集**七大智囊团**——张衡🔧、范蠡💰、韩非⚖️、商鞅⚔️、荀子📚、鬼谷子🧠、子贡📱——各抒己见，最终由她汇总裁决，输出方案。

**核心理念：单模型多视角**——通过 Prompt Engineering 在单一 LLM 内激活 7 种推理风格，零额外成本实现多智能体决策。

## 安装

### 方式一：通过 SkillHub 安装（推荐）

在 OpenClaw 中安装：

```
skillhub install hexi-sage-council
```

### 方式二：手动安装

```bash
git clone https://github.com/linlin-1107/hexi-7sages.git
cd hexi-7sages
bash scripts/install.sh
```

安装后运行 `openclaw gateway restart` 生效。

## 文件结构

```
hexi-7sages/
├── SKILL.md           — SkillHub 技能定义
├── scripts/

    ├── USER.md        — 用户画像模板（需自定义）
    └── MEMORY.md      — 长期记忆模板（需自定义）
```

## 设计理念

- **历史人物人设**：用古代贤士替代现代名人 Agent，规避法律风险，增强文化辨识度
- **零成本运行**：无需额外 API 调用，所有智囊共享同一模型的上下文窗口
- **渐进式裁剪**：非关键问题只召集2-3位相关智囊，重大决策才全员集结
- **完全开源**：MIT License，自由使用、修改、分发
